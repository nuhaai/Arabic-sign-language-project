# Arabic-sign-language-project
Develop an AI classification model to recognize Arabic Sign 
Language (ArSL) using computer vision and neural 
networks to support communication for the deaf and 
hard-of-hearing.

Technology Used
Deep learning with Convolutional Neural Networks 
(CNNs), including:
VGG16,
ResNet,
and MobileNet architectures.
